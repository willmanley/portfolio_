# <center> 52. N-Queens II </center>

https://leetcode.com/problems/n-queens-ii/description/

------------------------------------------------------------------------------------------------------------------------
## Problem Statement

The *n-queens* puzzle is the problem of placing n queens on an n x n chessboard, such that no two queens attack each
other.

Queens can "attack" by moving any number of cells: horizontally, vertically or diagonally.

Given an integer n, return the number of distinct solutions to the n-queens puzzle.


------------------------------------------------------------------------------------------------------------------------

## Constraints

- 1 <= n <= 9

------------------------------------------------------------------------------------------------------------------------

## Examples

### Example 1

```
>>> n = 4

< Expected Output: 2
```

### Example 2

```
>>> n = 1
< Expected Output: 1
```
